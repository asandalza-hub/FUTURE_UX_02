Salon Appointment Booking App UI (Figma)

This project is a mobile UI design for a salon appointment booking app, focused on creating a simple and seamless booking experience. The app allows users to browse services such as silk press, braids, hair steam, and blowouts, select a specialist, choose a date and time, and confirm their booking.

The design follows a clean, modern layout with an intuitive user flow from onboarding to confirmation, ensuring ease of use and clarity.

Created in Figma as part of a UX/UI design project to showcase mobile design, user experience, and interface structuring.
